RoboFriends application with ReactJs and Fetch API

  • Live Demo
  • Source code

This is a React Application called Robofriends where a user can search his friends in the search bar and the name will be filtered out dynamically according to the keywords which user will type.

  • This React application is made by using FetchAPI
  • The data is coming from the web services called jsonplaceholder
  • For the CSS and styling tachyons package is used.

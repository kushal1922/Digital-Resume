Random Paragraphs generator using Redux and Axios Libaries

  • Live Demo
  • Source code

This is a React Application where user can get one random Paragraph by clicking on a button. The paragraph will be fetched from one JSON file. Different paragraphs will be generated by changing the numbers from the input field.

  • This React application is made by using Redux and Axios Libaries.
  • The data is coming from the JSON file which contains the paragraphs with an unique ID.
